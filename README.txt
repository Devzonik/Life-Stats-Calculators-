== Description ==

Life Stats Pro is a high-performance, Elementor-compatible WordPress theme designed for life statistics and longevity insights. It features a real-time calculator that quantifies your journey through life in heartbeats, breaths, and more.

== Installation ==

1. Export the project as a ZIP from AI Studio (Settings -> Export to ZIP).
2. In your WordPress Dashboard, go to Appearance > Themes > Add New.
3. Click Upload Theme and select the ZIP file you downloaded.
4. Activate the theme.
5. Install and Activate the Elementor plugin for full page building capabilities.

== Features ==

* Real-time Life Stats Calculator on the homepage.
* 100% Elementor Compatibility.
* Responsive Design using Tailwind CSS.
* Customizer support for accent colors.
* SEO Optimized and Lightweight.
